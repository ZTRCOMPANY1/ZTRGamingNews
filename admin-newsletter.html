Como configurar

  1. Ative Authentication > Email/Password no Firebase.
  2. Crie o usuário administrador com o e-mail que você vai usar para entrar.
  3. Preencha assets/js/config.js com as chaves do seu projeto e o e-mail permitido em admin.allowedEmail.
  4. Publique as regras do arquivo firebase/firestore.rules.
  5. Se quiser domínio próprio, publique o site no GitHub Pages e continue usando o Firebase só para Auth/Firestore.
